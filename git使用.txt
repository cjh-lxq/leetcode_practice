初始化仓库
git init
添加远程托管服务器
git remote add [name] [url]
查看远程托管服务器
git remote -v
添加全部更改

git add -A
查看你添加了什么更改
git status
提交，一般要在添加更改后配套使用
git commit -m "something"
查看当前都有什么分支
git branch
添加新分支
git branch [branch_name]
改变当前分支(从一个分支切换到另一分支)
git checkout [branch]
删除本地分支
git branch -d [branch]
删除远程分支(原理是推送一个空分支到云端)
git push [url_name] :[branch]

推送分支到远程托管服务器上
git push [name] [branch]
强制更新到本地(慎用)
gir reset --hard orign master
下载项目的master分支到本地
git clone [url]
下载项目的指定分支到本地
git clone -b [branch] [url]
